# BFHL API

Spring Boot REST API developed for the Chitkara API Qualifier Round.

### Tech Stack

* Java 21
* Spring Boot
* Maven
* Docker
* Render

### Endpoints

* POST `/bfhl` – Processes the input array and returns the required response.
* GET `/health` – Health check endpoint.

### Hosted URLs

* https://bfhl-api-4y7k.onrender.com/bfhl
* https://bfhl-api-4y7k.onrender.com/health
